Criar Migration do zero

Add-Migration InitialCreate -Project OsLog.Infrastructure -StartupProject OsLog.Api -OutputDir EntityFramework\Migrations

Aplicar Migrations no Banco de Dados
Update-Database -Project OsLog.Infrastructure -StartupProject OsLog.Api



Apara Todas tabelas
dotnet tool run dotnet-ef -- database drop --force --project OsLog.Infrastructure --startup-project OsLog.API

Cria as tabelas a partir da Inicial
dotnet tool run dotnet-ef -- database update --project OsLog.Infrastructure --startup-project OsLog.API
